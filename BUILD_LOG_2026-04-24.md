## Build Task: Deep Research Skill

Created `skills/deep_research.py` - Multi-step research with synthesis inspired by DeerFlow 2.0's Deep Research capability.

### Features Implemented
1. **Source Credibility Assessment**
   - HIGH (0.85-0.95): Academic domains (.edu, .gov, arxiv.org)
   - MEDIUM (0.65): Established publications
   - LOW (0.3): Social media
   - UNKNOWN (0.5): Generic domains

2. **Multi-Step Iterative Research**
   - Configurable iterations (1-3)
   - Follow-up query generation
   - Entity extraction from titles

3. **Finding Synthesis**
   - Keyword-based clustering
   - Consensus confidence boosting (+0.1 per additional source)
   - Confidence cap at 0.95

4. **Knowledge Gap Identification**
   - Source coverage analysis
   - Temporal gap detection
   - Low-confidence flagging

5. **Report Generation**
   - JSON format (machine-readable)
   - Markdown format (human-readable with emoji indicators)

6. **Research History Management**
   - Query filtering
   - Confidence filtering
   - Cross-research comparison

7. **Tiered Memory Integration**
   - L1: Recent research queries
   - L2: High-confidence findings (>70%)

### Usage Example
```python
from skills.deep_research import create_deep_research_skill
from skills.web_search import WebSearchSkill
from core.tiered_memory import TieredMemorySystem

skill = create_deep_research_skill(
    web_search_skill=WebSearchSkill(),
    memory_system=TieredMemorySystem(agent_id="researcher")
)

report = skill.research("agi trends 2026", iterations=2)
print(f"Confidence: {report.overall_confidence:.0%}")
print(report.to_markdown())
```
