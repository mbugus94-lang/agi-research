# Safety Guidelines

## Core Rules

1. **No Auto-Apply** - System proposes, humans approve
2. **Sandboxed** - Experiments isolated
3. **Transparent** - All actions logged
4. **Reversible** - Easy rollback

## Self-Improvement Protocol

1. Propose → Write to proposals/
2. Review → Human approval
3. Test → Sandbox validation
4. Deploy → Merge after approval

## Forbidden
- Auto-modify source code
- Access secrets
- Uncontrolled network access
- Delete without backup

---
*Safety first, always.*
