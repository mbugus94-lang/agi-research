## Build Task: Embodied AI Simulation Layer

### Core Insight
Research from Meta's ARI acquisition and ARC-AGI-3 benchmark emphasizes that AGI requires physical world interaction and goal inference capabilities. Current agents operate in abstract symbolic spaces; embodied simulation provides:
- Physical reasoning and spatial understanding
- Goal inference from observations (ARC-AGI-3 style)
- Multi-modal perception (visual, spatial, relational)
- Action-effect learning through interaction
- Transfer to real-world robotics (sim-to-real)

### Implementation Features

#### 1. World Class
- Grid-based environment with configurable dimensions
- Cell types: EMPTY, AGENT, OBSTACLE, GOAL, OBJECT, INTERACTIVE
- Object placement and management
- Serialization/deserialization support

#### 2. EmbodiedAgent Class
- Perception system with visual grid, spatial relations, proprioception
- Action execution with transition recording
- BFS path planning with obstacle avoidance
- Action-effect learning from experience
- Goal inference from example pairs (ARC-AGI-3 style)

#### 3. PhysicsEngine Class
- Gravity simulation for affected objects
- Collision detection
- Movement tracking

#### 4. GoalInferenceTask Class
- ARC-AGI-3 style task definition
- Example-based learning
- Solution evaluation with similarity scoring

#### 5. Helper Functions
- `create_simple_navigation_task()`: Navigation benchmark
- `create_object_collection_task()`: Object collection benchmark

### Usage Example
```python
from core.embodied_simulation import (
    World, EmbodiedAgent, Action, ActionType, Position
)

# Create world
world = World(10, 10)
world.place_agent(Position(1, 1))
world.place_goal(Position(8, 8))

# Add obstacles
world.place_obstacle(Position(5, 5))

# Create agent
agent = EmbodiedAgent("explorer", world)

# Perceive environment
perception = agent.perceive()
print(f"Visual grid: {perception.visual_grid}")
print(f"Spatial relations: {perception.spatial_relations}")

# Plan and execute path
path = agent.plan_path(Position(8, 8))
for action in path:
    success, result = agent.act(action)
    print(f"Action: {action.action_type.name}, Success: {success}")

# Check learned knowledge
knowledge = agent.get_learned_knowledge()
print(f"Transitions recorded: {knowledge['transition_count']}")
```
